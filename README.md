# lobster-cli-hello-world

## Description
This is a POC project to demonstrate lobster, a cross between cpp and python.

Supports X11 display forwarding from docker container.

## Tech stack
- lobster
- gpp
- cmake

## Docker stack
- ubuntu

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
